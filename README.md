### Installation

1. Clone the repository
```bash
git clone https://github.com/Dalia-Ramadan/LoginUi.git
```

2. Navigate to the project directory
```bash
cd LoginUi
```

3. Get dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
